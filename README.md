# OpenWordandSearch

## 機能<br>
GoogleとWord(MicrosoftOffice)を起動します。<br>
それだけです。シンプルイズベストってことで。<br>

## 動かない場合<br>
・実行できない！<br>
→Python実行環境がない可能性があります。Pythonの実行環境を用意してください。<br>
・Wordが見つからない！<br>
→Wordのパスが違う可能性があります。確認し、違う場合はプログラム内「Wordのパス指定」の下にある''で囲まれているパスを書き換えてください。<br>
